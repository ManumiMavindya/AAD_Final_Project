# JobHub – Online Job Vacancy & Recruitment Management System

## 📌 Project Overview

JobHub is a web-based recruitment platform designed to connect job seekers and employers in a centralized digital environment.
The system allows companies to publish job vacancies and enables job seekers to search and apply for jobs easily.

This platform simplifies the recruitment process by providing a single system where employers can manage job postings and review applications while job seekers can explore career opportunities.

---

## 🎯 Objectives

* Provide an online platform for employers to publish job vacancies.
* Allow job seekers to search and apply for jobs easily.
* Simplify the recruitment and application management process.
* Improve communication between employers and applicants.

---

## 👥 System Users

### 1. Job Seekers

Job seekers can:

* Register and login to the system
* View available job vacancies
* Apply for jobs
* Upload their resume
* Track their job applications

### 2. Employers

Employers can:

* Register and login
* Create company profiles
* Post job vacancies
* Manage job listings
* View and manage applicants
* Send interview notifications

---

## 🧩 System Modules

### Authentication Module

Handles user registration and login.

Features:

* User registration
* User login
* Role-based access (Job Seeker / Employer)

---

### Company Management Module

Allows employers to manage company details.

Features:

* Create company profile
* Update company information
* View company details

---

### Job Management Module

Handles job vacancy management.

Features:

* Post job vacancies
* View job listings
* Update job details
* Delete job postings

---

### Job Application Module

Allows job seekers to apply for jobs.

Features:

* Apply for jobs
* View applied jobs
* Manage application status

---
